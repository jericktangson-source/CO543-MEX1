CO 543 – Computer Vision 
Machine Exercise 1 
Red Light Green Light 
Overview
This project implements a computer vision-based "Red Light Green Light" game using webcam input and motion detection. The system processes real-time video frames to determine whether a player is moving or staying still, and enforces game rules through a state machine.
The objective is to move during GREEN states and remain still during RED states. Violations result in warnings or termination (DEAD state).
